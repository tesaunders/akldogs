# Auckland Council Animal Management impound data

Data relating to impounded dogs taken from Auckland Council Animal
Management annual reports.

## Usage

``` r
impounds
```

## Format

A dataframe with 12 rows and 5 columns:

- year:

  Financial year data was reported

- impounds:

  Total number of dogs impounded

- returned:

  Total number of impounded dogs returned to or claimed by owner

- euthanised:

  Total number of impounded dogs euthanised

- transferred:

  Total number of impounded dogs adopted or transferred to a rescue
  centre
